# Cloudflare Pages Build Configuration

This file exists to ensure Cloudflare Pages uses the correct build settings.

## Build Settings

- **Build Command**: `bash build-web-client.sh`
- **Output Directory**: `frontend/apps/web-client/out`
- **Node Version**: 22.x
- **Package Manager**: npm

## Important Notes

- The repository uses npm workspaces
- Frontend apps are in `frontend/apps/`
- Backend code is in `backend/`
- Shared code is in `shared/`
