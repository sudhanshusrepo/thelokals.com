# Core

This package contains the core business logic and data models for The Lokals Platform. It is a library that is used by both the `client` and `provider` applications.

## Installation

To install the `core` package, you can use pnpm:

```bash
pnpm install @lokals/core
```

## Usage

Once you have installed the package, you can import the core modules in your code:

```typescript
import { Booking, Service, User } from '@lokals/core';

// ...
```
