# Documentation

Welcome to the `thelokals.com` documentation.

## Structure

- **[api/](./api/)**: API documentation and specifications.
- **[deployment/](./deployment/)**: Deployment guides, checklists, and summaries.
- **[development/](./development/)**: Architecture docs, coding standards, and roadmaps.
- **[features/](./features/)**: Feature specifications and design docs.
- **[packages/](./packages/)**: Package-specific documentation.
- **[reports/](./reports/)**: Implementation summaries, audits, and status reports.
- **[testing/](./testing/)**: Testing plans, checklists, and reports.
- **[troubleshooting/](./troubleshooting/)**: Guides for resolving common issues.

## Contributing

When adding new documentation, please place it in the most appropriate subdirectory. If a document spans multiple categories, consider cross-linking or placing it in `development`.
